Learning Python

For my first project I wanted to display my ability to teach myself new skills. I taught myself guitar, photography, photo editing, and other various skills. I chose to learn Python, after hearing about the language for awhile being praised as an easy to read format. However, while I certainly bit off more than I could chew for 20 hour work weeks, there were some valuable lessons learned from the experience:

  • Starting a new language means readin up on documentation for the language, any new softwares required, and any core packages that will be used. You learn more than just the language itself.
  • You shouldn't just download the latest version of a language if there is an entire level change. I downloaded Python 3.8, when I should have downloaded Python 3.7. This is because a language update is less about how it functions and more about how to use it. It was harder to find documentation on Python 3.8 for me to read and learn from.
  • I shouldn't set high expectations in limited time margins. I should have chosen something I was more comfortable with to display my actual strong suits as a programmer.
  • Lastly, Python was not object oriented it appeared and there was a new curveball thrown my way.

Regardless, I intend to keep learning Python but at a more relaxed pace so I can take the time to actually understand how each part actually works.
